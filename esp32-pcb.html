Custom ESP32 PCB — Learning Project

Project Overview

This project is a work in progress where I’m teaching myself PCB design by creating my own ESP32 board. The board includes a temperature sensor and is being designed as part of my learning process in hardware layout and embedded systems.

My Approach — Working Backwards

I’m working backwards from my existing temperature node used in my MQTT project. Instead of starting small, I began with a full ESP32 dev board design and I’m removing the parts I don’t need, like extra headers and unused features. The goal is to create a cleaner PCB focused only on my temperature node.

Connection to My MQTT System

This custom board will eventually replace my dev board and act as a dedicated temperature sensor node that sends data to my Raspberry Pi MQTT broker.
