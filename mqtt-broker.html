Architecture

  • ESP32 publishes sensor readings to topics (ex: sensor/room/temp_c)
  • Raspberry Pi runs Mosquitto broker
  • Python subscriber logs data (CSV/SQLite) and powers dashboards
  • Future: VPN access so I can monitor from anywhere

Useful commands (quick demo)

# install
sudo apt update
sudo apt install mosquitto mosquitto-clients -y

# start + enable
sudo systemctl enable mosquitto
sudo systemctl start mosquitto

# subscribe
mosquitto_sub -h <pi-ip> -t "sensor/#" -v

# publish test
mosquitto_pub -h <pi-ip> -t "sensor/test" -m "hello mqtt"
