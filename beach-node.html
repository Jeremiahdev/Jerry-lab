Beach Quality & Weather Node

ESP32 / Arduino sensor platform · part of environmental monitoring using drones

As part of an environmental monitoring initiative, our team designed sensor systems and drone concepts to monitor air, water, and soil around Caribbean beaches. This page shows selected slides from the project presentation — objectives, drone platforms, system diagrams, and future work — wrapped in my own portfolio styling.

Project Objective

Design an affordable sensor and drone system that can be used by students and researchers to collect environmental data (temperature, humidity, air quality, water quality, soil conditions) around beaches — without needing expensive commercial hardware.

High-level objective slide from the project deck.

Team & Roles

This work was done as a collaborative research project. Each student focused on a part of the system: drone design, sensor integration, simulation, control, and documentation.

Team overview slide, listing members and mentors.

HIPE 72 Neptune · Water Quality Drone

System Architecture & Node Design

The core idea is simple: sensors feed into an ESP32 or Arduino-based controller, which then sends data for logging and analysis. Below are some of the block diagrams and schematics used to plan the system.

Cost & Accessibility

One of the main constraints was cost. The budget slide breaks down how a multi-drone system can be built using off-the-shelf components, focusing on:

  • Airframe hardware and motors
  • Sensors and microcontrollers (Arduino / ESP32)
  • Radio/telemetry modules and batteries

Budget breakdown for hardware and components.

Tools, Simulation & Process

The project combined mechanical design, electronics, and simulation:

  • CAD and design tools for airframes
  • Arduino IDE / ESP32 environment for firmware
  • Simulation tools for flight and control tuning

Future Work

Next steps for the project include:

  • Building and flight-testing physical prototypes
  • Integrating live telemetry with web dashboards
  • Expanding sensor types (gas sensors, turbidity, etc.)
  • Deploying multiple nodes along a coastline for higher coverage

Planned improvements and extensions to the system.

What This Project Means to Me

This project is where my interests line up: embedded systems, IoT, and real-world impact. It pushed me to think not just about getting sensors to work, but about how students and communities could actually use a system like this to monitor the environment around them.
