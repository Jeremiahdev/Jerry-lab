Build steps

  1. Planned the layout: inputs (switches), IC area, outputs (LEDs), power.
  2. Mounted hardware: drilled the enclosure and installed switches + LEDs.
  3. Built the PCB / wiring: connected rails, inputs, and output indicators cleanly.
  4. Tested continuity: checked for shorts and confirmed power/ground paths.
  5. Verified logic behavior: inserted gate ICs and tested truth tables.
  6. Final assembly: secured the board inside the enclosure and confirmed stable operation.
note
The goal was a clean, reusable trainer that lets me test logic IC behavior quickly without rebuilding the circuit every time.

Testing / verification

  • Power check: confirmed 5V rail is stable.
  • Input check: each switch reliably produces LOW/HIGH states.
  • Output check: LEDs reflect output correctly (truth table testing).
  • Repeatability: swapped ICs and re-tested to confirm it‚Äôs consistent.
